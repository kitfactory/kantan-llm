# チュートリアル一覧

このフォルダには、単元ごとに学べるチュートリアルをまとめています。
各単元は「シーンの説明 → 使う機能の説明 → 使い方の説明 → コード」という順序で進みます。
順番に読み進める場合は 01 から始めてください。

01_getting_started.md では、最短セットアップと基本実行を確認します。
02_tracing_basics.md では、Trace/Span の概念と PrintTracer の表示を体験します。
03_sqlite_tracing.md では、SQLiteTracer による記録と簡単な検索を行います。
04_trace_search.md では、よく使う検索条件のパターンを整理します。
05_judge_loop.md では、judge ループの最短導線を確認します。
06_utilities.md では、補助ユーティリティの使いどころを理解します。
07_full_story.md では、記録 → 分析 → 改善 → 再評価の一連の流れを体験します。

前提として、Python 3.10 以上と `.venv` の仮想環境を利用します。
